# Augusta Tech Culinary Arts - To-Go Food Service Website

This is a one-page website project for the Augusta Technical College Culinary Arts Team's to-go food service, created for CIST1510 Web Development.

## Project Structure

```
project/
├── index.html           # Home page
├── contact.html         # Contact page
├── menu.html            # Full menu page
├── hours.html           # Hours and location page
├── about.html           # About the team page
├── README.md            # This file
├── css/
│   └── main.css         # Stylesheet for all pages
└── images/
    └── (add your images here)
```

## Features Implemented

✓ Home page with header, two columns (section and aside), and footer
✓ Images stored in `images/` directory
✓ CSS files stored in `css/` directory
✓ All HTML files in root directory
✓ Descriptive page names (contact.html, menu.html, etc.)
✓ Main content area with image and text
✓ Ordered list in main section
✓ Unordered list of links in aside
✓ At least 4 links to other pages in the aside
✓ No inline CSS - all styling in main.css
✓ Color scheme using Augusta Technical College colors (navy blue and gold)

## Color Scheme

- **Primary Color**: Navy Blue (#1f3c88) - Augusta Tech's official color
- **Accent Color**: Gold (#d4af37) - Complementary accent
- **Background**: Light Gray (#f4f4f4)
- **Text**: Dark Gray (#333)

## How to Use

1. Replace `images/culinary.jpg` with an actual food image from Unsplash or another source
2. Update content as needed while maintaining the structure
3. Test all links to ensure they work properly
4. When ready to submit, compress the entire folder and upload to Blackboard
